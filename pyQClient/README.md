## pyQClient

A lightweight Python desktop application using PyQt6 and HTML-based UI.


Create a virtual environment (optional but recommended):

```
//가상환경 설정
python -m venv venv

//가상환경 실행
.\venv\Scripts\activate
source venv/bin/activate 

//가상환경 종료
deactivate
```


```
pip install -r requirements.txt
```
